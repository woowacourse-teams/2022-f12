=== 제품 목록 페이지 조회

==== 발생 가능 예외

- 40000: 검색 조건이 유효하지 않은 경우
- 40000: 페이징 조건이 유효하지 않을 경우

==== 검색 조건

- 제품 이름을 query로 하여 검색
- 없을 경우 모든 제품 조회

==== category 조건
- `keyboard` (키보드)
- `mouse` (마우스)
- `monitor` (모니터)
- `stand` (거치대)
- `software` (소프트웨어)
- 없을 경우 모든 제품 조회

==== sort 조건
- `rating,desc`(평점순)
- `reviewCount,desc`(리뷰 많은순)
- 없을 경우 등록 순서
